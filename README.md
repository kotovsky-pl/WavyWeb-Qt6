# WavyWeb-Qt6

**WavyWeb-Qt6** is a lightweight web browser written in Python using **PyQt6**.  
It supports tabs, an address bar, and quick-access buttons to popular websites (DuckDuckGo, Google, YouTube).  

---

## Features

- Browser tabs (add, close, switch)  
- Address bar for entering custom URLs  
- Navigation buttons: Back ⬅, Forward ➡, Refresh ⟳  
- Quick-access buttons for popular sites: DuckDuckGo, Google, YouTube  
- History of visited websites (saved in `browser_data.json`)  

---

## Requirements

- Python 3.10+  
- PyQt6  
- PyQt6-WebEngine
